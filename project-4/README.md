# 🎯 General Knowledge Quiz (Python)

## 📌 Overview

This project is an **interactive command-line General Knowledge Quiz** built using Python. It asks users multiple-choice questions, validates input, tracks scores, and provides performance feedback at the end.

The quiz is designed to be **user-friendly, dynamic, and engaging**, with randomized questions and real-time feedback.

---

## 🚀 Features

* 🎯 Multiple-choice questions (A/B/C/D)
* 🔄 Randomized questions every run
* ⏱️ Timer for each question
* ✅ Input validation (prevents invalid answers)
* 📊 Score tracking and percentage calculation
* 🏆 Performance evaluation (Excellent, Good, etc.)
* 🔁 Replay option (play multiple times)
* 👤 Personalized experience using player name

---

## 🧠 Concepts Used

* Python Lists & Dictionaries

* Conditional statements (`if-elif-else`)

* Time tracking using `time` module
